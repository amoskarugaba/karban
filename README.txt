KARBAN - simple accounting
==========================

- WORK IN PROGRESS -
You will notice when it's ready ;)

This project aims to create a really basic accounting system for freelancers and small companies.

Its main goals are:
- Usability
- Straight forward, hackable design

KARBAN is written in PHP 5.3+ and published under Gnu General Public Licencse version 3. See LICENSE.

It uses ihumanable's prosper lib for database communication.
See http://github.com/ihumanable/prosper-lib for more information.
