KARBAN - simple accounting
==========================

- WORK IN PROGRESS -
You will notice when it's ready ;)

This project aims to create a really basic accounting system for freelancers and small companies.
KARBAN is written in PHP 5.3+ and published under GNU General Public Licence Version 3. 
See LICENCE.txt for details.

Its main goals are:
- Usability
- Straight forward, hackable design

It uses source code from the following projects:

- ihumanable's prosper lib for database communication.
See http://github.com/ihumanable/prosper-lib for more information.

- Adam Davis' userCake for user management
See http://usercake.com for more information.
